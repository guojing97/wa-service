Pengantar

Selamat datang di dokumentasi resmi **WA Service API**. REST API ini dirancang untuk memudahkan Anda berintegrasi dengan layanan WhatsApp, termasuk mengirim pesan teks, gambar, dan dokumen secara otomatis melalui pemrograman.

Layanan ini menggunakan arsitektur **Express.js** yang modern, dengan integrasi pustaka **Baileys** yang stabil. Semua endpoint menggunakan format respons **JSON** standar dan dilindungi oleh API Key demi keamanan.

  **Tips:** Untuk mencoba endpoint ini secara interaktif, gunakan halaman pengujian dashboard utama dengan mengklik tombol *Kembali ke Dashboard* di pojok kanan atas.

Autentikasi

Setiap permintaan ke API ini (kecuali endpoint `/health` dan `/api-config.js`) wajib menyertakan kunci autentikasi **x-api-key** di dalam header HTTP. Permintaan tanpa header ini atau dengan kunci yang salah akan menghasilkan respons `401 Unauthorized`.

  **Peringatan:** Amankan API Key Anda! Jangan menyebarkan kunci ini di repositori kode publik Anda.
Contoh Header HTTP: 
x-api-key: 4b9d8e7f2a1c6d5e3f0b9a7c4d2e8f1a6b3c9d7e5f2a8b1c4d6e9f7a3b5c2d1
Content-Type: application/json

API Sesi

Endpoint di bawah ini digunakan untuk melacak status koneksi WhatsApp Anda, menampilkan QR Code, dan mengontrol siklus sesi.

GET /api/session/status

Mendapatkan informasi detail mengenai status sesi koneksi WhatsApp saat ini.

Contoh Respons Sukses (200 OK): 
{
  "success": true,
  "message": "Session status retrieved",
  "data": {
    "status": "connected",
    "phone": "628123456789",
    "name": "My Business WA",
    "connectedAt": "2026-05-30T11:00:00.000Z",
    "retries": 0,
    "session": {
      "exists": true,
      "files": 12,
      "sizeHuman": "4.21 KB"
    }
  }
}
GET /api/session/qr

Mendapatkan data QR Code dalam format **Base64 Data URL** untuk dipindai di WhatsApp.

Contoh Respons (200 OK): 
{
  "success": true,
  "message": "Scan this QR code with WhatsApp",
  "data": {
    "status": "qr",
    "qr": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgA..."
  }
}
POST /api/session/logout

Memutuskan sesi WhatsApp saat ini dan **menghapus semua file otentikasi** di server.

API Pesan

Endpoint ini digunakan untuk mengirim berbagai bentuk pesan WhatsApp. WhatsApp **harus berada dalam kondisi terhubung (connected)** untuk menggunakan endpoint ini.

POST /api/message/send-text

Mengirimkan pesan teks biasa atau pesan teks yang menggunakan emoji.

Parameter Tipe Kewajiban Deskripsi
phone String Wajib Nomor WhatsApp tujuan (contoh: 08123456789 atau 628123456789).
message String Wajib Konten teks pesan yang ingin dikirimkan.
Contoh Command cURL: 
curl -X POST http://localhost:3000/api/message/send-text \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-secret-api-key" \
  -d '{
    "phone": "628123456789",
    "message": "Halo, ini pesan pengujian otomatis! 🚀"
  }'
POST /api/message/send-image

Mengirimkan pesan gambar menggunakan URL publik atau representasi data base64.

Parameter Tipe Kewajiban Deskripsi
phone String Wajib Nomor WhatsApp tujuan.
image String Wajib URL gambar publik (HTTP/HTTPS) ATAU string data base64 dari gambar.
caption String Opsional Teks teks keterangan gambar yang dilampirkan.
Contoh Command cURL: 
curl -X POST http://localhost:3000/api/message/send-image \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-secret-api-key" \
  -d '{
    "phone": "628123456789",
    "image": "https://picsum.photos/800/600",
    "caption": "Pemandangan hari ini! ☀️"
  }'
POST /api/message/send-document

Mengirimkan file dokumen (seperti PDF, Excel, Word, dll) via WhatsApp.

Parameter Tipe Kewajiban Deskripsi
phone String Wajib Nomor WhatsApp tujuan.
document String Wajib URL dokumen publik (HTTP/HTTPS) ATAU string data base64.
filename String Wajib Nama file saat diterima oleh WhatsApp penerima (misal: `invoice.pdf`).
mimetype String Opsional Mime-type file. Default: `application/pdf`.
Contoh Command cURL: 
curl -X POST http://localhost:3000/api/message/send-document \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-secret-api-key" \
  -d '{
    "phone": "628123456789",
    "document": "https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf",
    "filename": "laporan_keuangan.pdf",
    "mimetype": "application/pdf"
  }'
